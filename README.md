# Java Binary Conversion Assignment

This repository contains solutions for a Programming Fundamentals assignment.

## 📌 Objective
To convert a decimal (integer) number into its binary representation using different approaches in Java.

---

## 📂 Contents

### 🔹 Part (a)
A Java program that:
- Takes an integer input from the user
- Converts it into binary using a loop
- Displays the result

---

### 🔹 Part (b)
A user-defined function that:
- Takes an integer as a parameter
- Converts it into binary
- Stores the binary digits in an integer array
- Returns the array

---

### 🔹 Part (c)
A user-defined function that:
- Takes an integer as a parameter
- Converts it into binary
- Stores the binary digits in a string
- Returns the string

---

## 🧪 Test Cases

| Decimal | Binary |
|--------|--------|
| 32     | 100000 |
| 27     | 11011  |
| 26     | 11010  |
